Duplicate posts and pages, including custom post types.

== Description ==

Duplicate posts and pages. Works seamlessly with custom post types and Advanced Custom Fields!

Major features in WP Duplicate Posts And Pages include:

* Supports Custom Post Types
* Supports Advanced Custom Fields

== Installation ==

Upload WP Duplicate Posts And Pages folder to your wp-content/plugins directory, then Activate it. No configuration needed.
